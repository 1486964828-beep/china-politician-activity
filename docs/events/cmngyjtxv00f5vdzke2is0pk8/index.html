陈书平赴成华区、彭州市、成都高新区调研并走访重点企业开展进解优促时强调 以一流营商环境护航企业高质量发展

调研成都2026-03-27

摘要

近日，市委副书记、市长陈书平先后前往成华区、彭州市、成都高新区，深入重点企业走访调研，现场开展进万企、解难题、优环境、促发展工作，强调要深入学习贯彻习近平总书记在全国两会期间的重要讲话和全国两会精神，认真落实党中央、国务院决策部署，以及省委、省政府和市委、市政府工作要求，着力提升惠企政策精准度、为企服务实效性，以一流营商环境护航企业高质量发展，为奋力实现一季度 开门红、推动全市经济高质量发展提供坚实支撑。听取成华区经济社会发展情况汇报后，陈书平强调——成华区要咬定目标不放松，立足区位条件和基础优势，在产业建圈强链、深化科产融合、拓展开放合作、城区功能品质提升上创新突破，在全市高质量发展大局中多作贡献
